Step 01

乐谱格式转化

使用卷积网络或视觉模型，把五线谱图片、扫描件或结构化乐谱转成后续模型可用的 Python 数据格式。

Step 02

和声与旋律生成

在已有旋律轮廓、和声条件与风格参数上，调用多层神经网络或外部 AI API 生成新的即兴乐段。

Step 03

试听与交互输出

站点端负责把结果转成可预览、可试听、可导出的交互界面，并为后续部署和接口联调留好位置。
